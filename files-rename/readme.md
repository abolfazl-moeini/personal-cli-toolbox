Rename downloaded courses from coursehunter to human readable file names.

the default files would be 
`
lesson1.mp4
lesson2.mp4
...
`

and video titles in separate txt file

`
Welcome
Setup environment
...
`

After running 
```bash
cd {video-folder}
./files-rename.sh
```

New files are
`
01-Welcome.mp4
01-Setup Environment.mp4
...
`
